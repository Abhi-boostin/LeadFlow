# LeadFlow

A single-screen, AI-augmented CRM for sales reps. Track leads, log every discussion, set follow-up reminders, and let AI handle the heavy lifting on meeting notes.

## Status
In active development. Setup steps and a live demo URL will land here as the scaffold comes together.

## Stack
- **Frontend:** Next.js 14 (App Router) on Vercel
- **Backend:** Fastify + TypeScript on Render
- **Database:** PostgreSQL on Supabase
- **AI:** Groq (Whisper transcription, Llama 3.3 70B summarisation)
- **Auth:** Google OAuth via Auth.js
- **Notifications:** Fast2SMS (SMS), Resend (email)
